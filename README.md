# playtime
Organise your movie collection using movie metadata and directory symlinks. Works with any mediacenter.

## What
Playtime is a utility to help identify and organise your collection of movies on your server or NAS.

Playtime uses IMDB data to identify movies and to get metadata. It then uses that metadata to construct a directory hierachy with symbolic links to the movies organised by genre, year, actor, director, and more. It can also download cover images for the movies to make browsing them even more fun.
